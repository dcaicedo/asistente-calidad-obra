# Asistente de Calidad en Obra

Herramienta de consulta rápida del **Manual de Estándares de Calidad** para el equipo de obra.

Funciona **sin internet**, directo desde el celular o tablet, sin instalar ninguna app.

---

## Cómo usarla

1. Abre el link en tu celular
2. Escribe lo que vas a hacer — con tus propias palabras
3. La app te muestra qué estándares de calidad aplicar, qué problema evitas y qué acción tomar

**Ejemplo:** escribe *"voy a asentar ladrillos cerca del tablero eléctrico"* y la app te responde con el estándar correspondiente.

---

## Contenido

- 27 estándares de calidad extraídos del Manual oficial
- Cubre las fases: Acabados Húmedos, Acabados Secos, Casco, Casco (Prelosas) y Muro Pantalla
- Búsqueda por relevancia con porcentaje de coincidencia
- Cada resultado muestra: qué hacer, qué pasa si no lo haces, y el estándar oficial

---

## Cómo abrir la app

**Desde el celular:**
Abre este link en el navegador (Chrome o Safari):

```
https://dcaicedo.github.io/asistente-calidad-obra/asistente_calidad.html
```

> Reemplaza `TU-USUARIO` con tu nombre de usuario de GitHub.

**Sin internet:**
Descarga el archivo `asistente_calidad.html` y ábrelo desde el celular. Funciona completamente offline.

---

## Cómo actualizar los estándares

1. Actualiza el Excel del manual con los nuevos estándares
2. Sube el Excel al chat con el asistente IA que generó esta app
3. Pide que regenere `asistente_calidad.html` con los nuevos datos
4. Sube el nuevo archivo aquí en GitHub con el mismo nombre
5. El link se actualiza automáticamente para todos
